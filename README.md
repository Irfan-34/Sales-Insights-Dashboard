# 📊 Sales Insights Dashboard (Power BI)

## 🔍 Overview

This project is an interactive Sales Dashboard built using Power BI to analyze business performance, identify trends, and gain insights into product sales and customer behavior.

## 🎯 Objectives

* Analyze sales trends over time
* Identify top-performing products
* Understand customer purchasing patterns
* Enable dynamic filtering using slicers

## 📈 Features

* KPI Cards (Total Sales, Total Products Sold, Average Sales, Sales per Customer)
* Time-based Sales Trend Analysis (Year-Month)
* Top 5 Products by Sales
* Interactive Slicers for filtering (State, Date)
* Insight section highlighting key business findings

## 🛠️ Tools & Technologies

* Power BI
* DAX (Data Analysis Expressions)
* Data Modeling

## 🧠 Key Insights

* Sales peaked around mid-2016
* Top 5 products contribute major revenue
* Sales declined after peak period
* Some recovery observed in 2018

## 📌 Conclusion

The dashboard provides a clear and interactive way to monitor sales performance and supports data-driven decision making.
